[02] Curriculum_Map

The Learning Journey

Three sessions, three breakthroughs. Each class builds on the last, taking you from blank screen to sensor-driven installations.

01
> Session_One

Hola Mundo (Hello World)

→ A custom interactive system that responds to your touch

Move from a "blank screen" to a living, breathing particle system using p5.js fundamentals and real-time interaction. You'll understand how code thinks—and how to make it dance.

02
> Session_Two

Hola Yo (Hello Me)

→ Audio-reactive visuals that transform sound and motion into art

Use your voice and image as a brush by implementing camera/microphone feeds and mapping movement to visual distortion. Your body becomes the controller.

03
> Session_Three

Hola Bichos (Hello Bugs)

→ A 'digital bug' system controlled by physical sensors

Break the screen barrier. Learn Arduino integration and serial communication to manipulate code with physical objects like light and distance sensors.
